# Språkkafé Oslo

Directorio comunitario Mobile First para encontrar actividades gratuitas donde practicar noruego, conocer personas y conectar con la vida local en Oslo.

## Desarrollo

Requiere Node.js 22.

```bash
npm install
npm run dev
```

## Validación

```bash
npm run lint
npm run build
```
