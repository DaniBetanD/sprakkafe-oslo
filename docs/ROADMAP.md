# Visión

Språkkafé Oslo no pretende ser únicamente un directorio de actividades.

Nuestro objetivo es construir la plataforma de referencia para la integración lingüística y social de personas inmigrantes en Noruega.

Comenzamos en Oslo.

Pero el proyecto está diseñado para poder crecer a todo el país.

Cada nueva funcionalidad debe acercarnos a esa visión.

# Objetivo del MVP

Cuando finalice el MVP el usuario deberá poder:

✓ Encontrar una actividad fácilmente.

✓ Entender qué es un Språkkafé.

✓ Sentirse seguro para asistir.

✓ Descubrir organizaciones fiables.

✓ Navegar cómodamente desde el móvil.

Si conseguimos esto, el MVP estará completo.

---

# Indicadores de éxito

El proyecto considerará completada su primera fase cuando:

□ Toda la navegación sea intuitiva.

□ El diseño sea consistente.

□ El rendimiento sea excelente.

□ La web sea completamente responsive.

□ La información esté verificada.

□ La actualización de contenidos sea sencilla.

□ El usuario encuentre una actividad en menos de dos minutos.

---

# Filosofía del Roadmap

No construimos funcionalidades por construir.

Cada nueva característica debe responder al menos a una de estas preguntas:

• ¿Hace más fácil encontrar actividades?

• ¿Genera más confianza?

• ¿Reduce la fricción?

• ¿Ayuda a crear comunidad?

Si la respuesta es no...

...esa funcionalidad puede esperar.

---

# Visión a largo plazo

Queremos que, dentro de unos años, cualquier persona que llegue a Noruega pueda abrir Språkkafé, encontrar una comunidad cerca de ella y dar su primer paso para sentirse parte de este país.

Ese sigue siendo el objetivo principal del proyecto.
